# Ant_Colony_Optimization

代码学习参考于->https://zhuanlan.zhihu.com/p/137408401

这次是人工智能实验的结课作业啦。

写了一个*ant*的类，包含：

>clean_data  
>select_next_city  
>add_distance  
>lets_go

一共四个方法。

ant类是代码的灵魂之一，另一个灵魂是后半部分的**信息素矩阵更新**的部分了。

最后利用*matplotlib*库做了一个简单的可视化。
